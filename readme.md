# Budget Tracker CLI

A personal command-line budget tracking tool written in Python. Tracks monthly spending against a set budget, logs transactions, and calculates your remaining balance.

## Features

- Set a monthly budget
- Log expenses and income
- View remaining balance and days until payday
- View transaction history
- Reset budget and transactions

## Usage

Run with python main.py and follow the prompts. Available commands:

- set — set your monthly budget
- expense — log a debit transaction
- income - log money coming into budget
- balance — view remaining balance
- history — view transaction history
- reset — clear budget and transactions
- exit — save and quit

## Files

- amount.txt — stores your monthly budget
- transactions.txt — stores transaction history (excluded from version control)

### Requirements

Python 3.x — no external dependencies
